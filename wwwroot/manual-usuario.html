Sistema de Gestion Comercial COMARA - Guia completa de uso

Introduccion

Que es COMARA?

COMARA es un sistema integral de gestion comercial diseñado para empresas que necesitan administrar:

  • Inventario: Control de stock de productos
  • Ventas: Registro y seguimiento de ventas
  • Cotizaciones: Generacion de presupuestos
  • Facturacion: Integracion con AFIP
  • Caja: Control de movimientos de dinero
  • Clientes/Proveedores: Gestion de relaciones

Requisitos del Sistema

  • Navegador web moderno (Chrome, Firefox, Edge)
  • Conexion a internet (requerida para facturacion electronica)

Acceso al Sistema

Inicio de Sesion

  1. Abra su navegador web
  2. Ingrese la direccion del sistema
  3. Complete los campos: Usuario y Contraseña
  4. Haga clic en "Iniciar Sesion"

Cerrar Sesion

  1. Haga clic en su nombre de usuario (esquina superior derecha)
  2. Seleccione "Cerrar Sesion"

Cambiar Contraseña

  1. Haga clic en su nombre de usuario
  2. Seleccione "Cambiar Contraseña"
  3. Ingrese la contraseña actual
  4. Ingrese la nueva contraseña (dos veces)
  5. Haga clic en "Guardar"

Modulo de Cotizaciones

Acceso: Menu Principal → Cotizaciones

Que es una Cotizacion?

Una cotizacion es un presupuesto que se entrega al cliente. A diferencia de las ventas:

  • No afecta el stock
  • No se puede facturar directamente
  • Puede convertirse en venta cuando el cliente acepta

Estados de Cotizacion

  • Pendiente - Esperando respuesta del cliente
  • Aprobada - Cliente acepto el presupuesto
  • Rechazada - Cliente no acepto

Convertir Cotizacion a Venta

Cuando el cliente acepta la cotizacion:

  1. Vaya a la lista de cotizaciones
  2. Encuentre la cotizacion aprobada
  3. Haga clic en "Convertir a Venta"
  4. El sistema crea una nueva venta con los mismos datos
  5. La nueva venta queda lista para facturar

Acceso: Menu Principal → Facturas

Factura B6Para consumidores finales o monotributistas
Factura C11Emitida por Monotributistas
Nota de Credito A3Anulacion/descuento para Factura A

Generar Factura desde una Venta

  1. Vaya a Ventas
  2. Encuentre la venta a facturar (sin CAE)
  3. Haga clic en "Ver Detalles"
  4. Haga clic en "Generar Factura"
  5. El sistema valida los datos del cliente
  6. Seleccione el Tipo de Comprobante apropiado
  7. Revise los datos mostrados
  8. Haga clic en "Autorizar"
  9. El sistema se conecta con AFIP
  10. Si es exitoso, recibira el CAE y Numero de Comprobante
Importante:
  • Para Factura A: requiere CUIT valido del cliente
  • Para Factura B: acepta DNI o consumidor final

Modulo de Precios y Listas

Listas de Precios

Acceso: Menu Principal → Listas

Que es una Lista de Precios?

Las listas de precios permiten manejar diferentes tarifas para diferentes tipos de clientes. Cada lista tiene un porcentaje de margen que se aplica sobre el costo del producto.

Ejemplo:

  • Lista "Minorista": 50% de margen
  • Lista "Mayorista": 30% de margen
  • Lista "Especial": 20% de margen

Gestion de Precios

Acceso: Menu Principal → Precios

Ver Precios

La pantalla muestra todos los articulos con sus precios en cada lista (L1 a L5).

Aumentar Precios Masivamente

  1. Vaya a Precios → Aumentar Precios
  2. Seleccione el criterio: Por Marca o Por Proveedor
  3. Seleccione la marca o proveedor
  4. Ingrese el Porcentaje de Aumento (ej: 10 para 10%)
  5. Haga clic en "Aplicar"
El sistema actualiza automaticamente:
  • Todos los precios de listas (L1 a L5)
  • El costo del articulo

Descuento de Proveedor

El campo Descuento es importante porque afecta el calculo de costos:

Costo Final = Costo x (1 - Descuento/100)

Ejemplo: Si un articulo cuesta $100 y el proveedor da 10% de descuento:

Costo Final = $100 x (1 - 10/100) = $100 x 0.90 = $90

Flujos de Trabajo Comunes

Flujo 1: Venta Completa con Facturacion

1. Crear Cliente (si es nuevo) → Menu → Clientes → Crear Nuevo → Completar datos (especialmente CUIT/DNI y Condicion IVA) 2. Crear Venta → Menu → Ventas → Crear Nueva → Seleccionar cliente → Agregar articulos → Guardar 3. Facturar → Ver detalles de la venta → Generar Factura → Seleccionar tipo de comprobante → Autorizar con AFIP 4. Registrar Cobro (si es necesario) → Menu → Caja → Registrar Pago → Seleccionar cliente y monto

Flujo 2: Cotizacion que se Convierte en Venta

1. Crear Cotizacion → Menu → Cotizaciones → Crear Nueva → Seleccionar cliente, agregar articulos → Estado: Pendiente → Guardar 2. Entregar presupuesto al cliente 3. Si el cliente acepta: → Editar cotizacion → Estado: Aprobada → Convertir a Venta 4. Facturar la nueva venta

Flujo 3: Control de Stock

1. Configurar stock minimo de articulos → Menu → Stock → Editar cada articulo → Establecer stock minimo 2. Revisar alertas periodicamente → Ver notificacion en menu Reportes → Ir a Reporte de Stock Critico 3. Reponer stock → Contactar proveedor → Actualizar stock cuando llegue mercaderia → Menu → Stock → Editar → Actualizar cantidad

Flujo 4: Aumento de Precios

1. Ir a Precios → Aumentar Precios 2. Seleccionar criterio: → Por Marca (todos los productos de esa marca) → Por Proveedor (todos los productos de ese proveedor) 3. Ingresar porcentaje (ej: 15 para 15%) 4. Aplicar 5. Verificar en Precios → Index

Flujo 5: Cierre de Caja Mensual

1. Ir a Caja → Index 2. Seleccionar mes y año 3. Revisar: → Efectivo en caja → Cheques pendientes → Cuentas corrientes 4. Verificar balance por metodo de pago 5. Exportar o imprimir si es necesario
